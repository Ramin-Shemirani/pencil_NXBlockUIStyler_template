Description

The Pencil NX Block UI Styler Collection is a UI component collection, that enables NXOpen API Programers to quickly prototype Block UI Styler based dialogue boxes.

Goal

Use Pencil as the defacto standard Prototyping Tool for state of the art NXOpen UI Designs.

Prerequisites:

This collection was developed for the use with Pencil Project. It was initiated under Pencil V3 Beta.

RoadMap

Repo init - Repository name, description and content defined v0.0.0.1 alpha - small collection with minimum logic Available components Dialogue, Tabs, Groups, Subgroups, ActionButton, OK, APPLY, CANCEL, String Input
